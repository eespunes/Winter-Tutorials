Exercise Info

In order to create a Test exercise type, you should enter the question and the answers separated,
the questions surrounded by a #Q #/Q block, and the answers by a #R #/R block, adding a #C to the correct solution.
Example:
#QIn Java, the variable type that only has two possible values is:#/Q
#R The boolean#C#/R
#R The int#/R
#R The String#/R
#R The double#/R
Being "boolean" the correct answer, we add the #C command to indicate it.
