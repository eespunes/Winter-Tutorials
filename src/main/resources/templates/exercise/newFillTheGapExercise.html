Exercise Info

In order to create a "Fill the blank" exercise type, you should enter all the text(with the solution included) and surround the solution by a #R #/R block.
Example: In Java, the variable type that only has two possible values is a #Rboolean#/R.
Being "boolean" the word that the users must put on the gap to complete the exercise.
